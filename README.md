# FE25-JS2-slutprojekt-front-Mohsen-Heidari

Detta är frontend-delen av projektet Scrum Board. Frontend är byggd med Vite, TypeScript och Vanilla JavaScript och används för att visa och hantera uppgifter och medlemmar genom backend-API:et.

Applikationen visar en Scrum board med tre kolumner:

New – nya uppgifter

Doing – uppgifter som håller på att göras

Done – färdiga uppgifter

Användaren kan skapa medlemmar, skapa uppgifter, tilldela uppgifter till medlemmar och uppdatera uppgifter tills de är klara.

Frontend kommunicerar med backend genom HTTP requests (fetch).
